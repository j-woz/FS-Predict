  Path to the saved MLP model serialized with `joblib`.

* `-k scaler_path=../saved_models/scaler.pkl`  
  Path to the saved `MinMaxScaler` used during model training.

* `-s ../tmp/sus/sock.s`  
  Unix domain socket file path for server-client communication.
